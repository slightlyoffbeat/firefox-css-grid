# Firefox + CSS Grid (WIP)

A short intro to CSS Grid and Firefox CSS Grid Layout Panel

## Getting Started

This is a work-in-progress.


### Installing

If you want to run locally with CSS live-reloading, `npm install` and then run:

```
npm run dev
```
